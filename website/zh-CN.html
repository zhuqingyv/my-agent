MA

本地优先、多模型的终端编程 Agent。

在 LM Studio 和 DeepSeek 之间切换，让 API Key 离开明文配置，并在真实仓库里使用 MCP 工具完成工作。

/model  用上下键切换 provider 和模型

面向日常开发

尊重真实项目的终端体验。

默认多模型

当前支持 LM Studio 本地模型和 DeepSeek。Provider/model profile 让同名模型也能清楚区分：DeepSeek/deepseek-v4-pro。

快速切换模型

/model 打开键盘选择器，聚合发现到的模型，并记住上一次选择的 profile。

远程密钥更安全

远程 API Key 存在 macOS Keychain。配置文件只保存引用，不保存明文密钥。

内置 MCP 工具

ma init 后即可使用 shell、文件读写、结构化编辑、grep 和 web search。

斜杠指令

输入 / 查看建议，用上下键选择，用 Tab 补全。

项目指令和技能

MA 读取 AGENT.md 和项目内 .ma/skills/*.md，把团队工作流沉淀到仓库里。

Alpha Benchmark

本地 Qwen3-30B 通过 L0-L2 发布门槛。

MA 的 alpha benchmark 覆盖连接稳定性、工具调用稳定性和多轮本地项目任务。它是产品发布门槛，不是通用排行榜。
